Titanic Survival Prediction

A classic ML rite of passage: predicting survival on the infamous Titanic dataset from Kaggle.

View on GitHub

Project Goal

The aim was to predict whether a passenger survived the Titanic disaster, using demographic and ticket data such as age, gender, class, and fare.

The Brief & Challenge

This dataset is small but iconic, perfect for practicing feature engineering and classification. The challenge was to handle missing values, engineer useful features, and build models that could generalize well despite limited data.

Process & Outcome

Steps included cleaning the data, handling missing ages, encoding categorical variables, and performing exploratory analysis to find survival patterns. Multiple classification algorithms such as Logistic Regression, Random Forest, and Support Vector Machine were tested.

The final model achieved solid accuracy and confirmed some well-known patterns—such as higher survival chances for women and first-class passengers.
